Software Engineering - Axiom Zen

One of my most proud moments was the launch of Epics. Epics is a common concept in project management. It is essentially a collection of project management tasks (in this case, GitHub Issues). This was one of the biggest features that ZenHub had release to date, and since I was one of the main developers on this feature, it was great to see it go live.

During my work at ZenHub, I had a chance to work with reactive programming and a virtual DOM library, which further expanded my knowledge of different types of front end architecture. I worked on the front end display of the coordinates and images using the Google Maps API and Socket IO. More information can be found here.

Software Engineering Intern - Nexj Systems
May 2015 - August 2015

I was apart of the Nexj Health team working on a product called Connected Wellness, a wellness solution using wearable technology.

In order to better understand patients, Connected Wellness allowed "Assessments" to be assigned by a health care professional (HCP) so that they can better evaluate the patient's current health. Assessments are pre- and post- clinical surveys that are given to a patient to increase the efficiency of the appointment. Assessments would be scored to determine what state the individual's health is in, and then added to a group according to the results. I mainly worked on the team responsible for making an those assessments. I also implemented most of the scoring logic for the assessments.

In addition to the assessments, I was able to modify and add mobile responsive features in other parts of the app.

At Nexj Health, I also spent some time fixing server side issues. I used tools such as Kibana, Fiddler and the Node Inspector to analyze data and various http requests. In order to sufficiently test the security of the app, I needed to intercept http requests, modify the request, and resend it in order to try to use the app in ways it wasn't meant to be used.

Tools used: JavaScript, NodeJS, MongoDB, Less, Mercurial, SOA, Vagrant, Kibana, Fiddler
